# Examples / Legacy

Esta carpeta contiene versiones alternativas y código legacy del proyecto.

## main_console.py

Versión standalone con interfaz de consola del grabador de mouse. Esta es la versión original antes de implementar la interfaz gráfica con Streamlit.

**Uso:**
```bash
python examples/main_console.py
```

Esta versión es independiente y no requiere Streamlit, pero tiene menos funcionalidades que la versión principal con interfaz gráfica.
